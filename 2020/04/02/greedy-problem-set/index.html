5. HDU1338 游戏预测

【题目描述】

假设有M个人，包括你，在玩一个特殊的纸牌游戏。一开始，每个玩家得到N张牌。一张牌的pip是一个正整数，它的最大值是N * M。没有两张牌是相同的。在一轮游戏中，每个玩家选择一张牌与其他牌进行比较。拥有最大pip的牌手赢了这一轮，然后下一轮开始。经过N轮后，当每个玩家的所有纸牌都被选中时，赢得最多回合的玩家就是游戏的赢家。
给定你在开始时收到的牌，编写一个程序来告诉你在整个游戏中至少可以赢的最大轮数。

【输入】

输入由几个测试用例组成。每个case的第一行包含两个整数m (2 <= m <= 20)和n (1 <= n <= 50)，分别表示玩家人数和每个玩家在游戏开始时收到的牌数。然后是一行n个正整数，表示你在开始时收到的纸牌的点数。然后是空行来分隔这些情况。
输入以带有两个0的行结束。

【输出】

对于每个测试用例，输出一行由测试用例号和至少在游戏中获胜的轮数组成的代码。

【样例输入】
2 5
1 7 2 10 9

6 11
62 63 54 66 65 61 57 56 50 53 48

0 0
【样例输出】
Case 1: 2
Case 2: 4

【解析】
题目大意：求至少能赢的轮数
贪心策略：寻找恰好大于自己的数来对战，剩下的都能赢

【代码】

#include

#include

{
    int n, m, s, k, i, ans,j=0;

【题目描述】
约翰要去钓鱼。他有h个小时的时间(1 <= h <= 16)，在这片区域有n个湖泊(2 <= n <= 25)，都可以沿着一条单行道到达。约翰从第一个湖开始，但他可以在任何他想去的湖结束。他只能从一个湖到下一个湖。他不必停在任何一个湖上，除非他愿意。对于每个i = 1，…，n - 1，从i湖到i + 1湖所需要的5分钟间隔数记为ti (0 < ti <=192)。例如，t3 = 4表示从3湖到4湖需要20分钟。为了帮助计划他的钓鱼之旅，约翰收集了一些有关湖泊的信息。对于每个i湖，初始5分钟内预计捕获的鱼的数量(记为fi(fi >= 0))是已知的。5分钟内可捕获的鱼的数量每个间隔减少di (di > = 0)。如果鱼的数量预计将在一个时间间隔小于或等于di, 在接下来的时间间隔，湖中不会有鱼。为了简化计划，约翰假设，没有其他人会在湖上捕鱼来影响他希望捕捉到的鱼的数量。
编写一个程序，帮助约翰计划他的钓鱼之旅，以最大限度地增加鱼的期望被捕获的数量。在每个湖上花费的分钟数必须是5的倍数。

【输入】

输入有多个样例。每个样例第一行包含n。第一行包含h。接下来,有n个整数代表 fi (1 <= i <= n),然后一行n个整数 di (1 <= i <= n),最后一行 n - 1整数ti(1 <= i <= n - 1)输入终止的一个案件中,n = 0。

【输出】

对于每个测试用例，打印在每个湖上花费的分钟数(用逗号分隔)，以获得期望捕获的最大鱼类数量的计划(即使超过80个字符，也应该将整个计划打印在一行上)。接下来是一行包含期望的鱼的数量。
如果存在多个计划，选择一个尽可能长时间呆在一号湖的计划，即使预计在某些时间间隔内不会捕到鱼。如果仍然有多个计划可选择，选择一个花尽可能长的时间在湖2，依此类推。样例间之间插入空白行。

【样例输入】

2
1
10 1
2 5
2
4
4
10 15 20 17
0 3 4 3
1 2 3
4
4
10 15 50 30
0 3 4 3
1 2 3
0

【样例输出】
45, 5
Number of fish expected: 31

240, 0, 0, 0
Number of fish expected: 480

115, 10, 50, 35
Number of fish expected: 724
【解析】
题目大意：求h时间内能钓的鱼的最大数目
贪心策略：总时间= 钓鱼时间+走路时间. 所以我们枚举约翰最后钓鱼的湖是第i个,然后我们算出他一共花了多少时间走路,那么剩下的就是钓鱼花的时间.
求1-i钓的鱼的最大数目，在每单位时间内贪心选择能钓最多的鱼的湖泊，虽然顺序不一样，但是实际并不影响（思考一下）。
做法：枚举结束湖泊，预处理出走路时间，用优先队列保存并更新湖泊的状态，从而贪心选择湖泊。

【代码】

#include 
#include 
#include 

using namespace std;
const int maxn = 30;
int n,h;
int f[maxn], d[maxn], t[maxn], st[maxn];

struct Node{
    int i,f,t;
    Node(int i=0, int f=0, int t=0): i(i), f(f), t(t){}
    bool operator < (const Node &a)const{
        if(this->f == a.f) return this->i > a.i;
        else return this->f < a.f;
    }
};

priority_queue > pq;

int main()
{
    while(cin >> n&& n)
    {
        cin >> h;
        f[0] = d[0] = t[0] = st[0] = 0;
        for(int i = 1; i <= n; i++) cin >> f[i];
        for(int i = 1; i <= n; i++) cin >> d[i];
        for(int i = 1; i < n; i++) cin >> t[i];
        for(int i = 1; i <= n; i++) st[i] = st[i-1] + t[i-1];
        int ans = -1; // 记录最优解，有可能没有鱼，所以初始值需要设置为负数
        int ret[maxn]; //记录答案分钟数
        for(int i = 1; i <= n; i++)
        {
            int res = 0; // 记录改方案总鱼数目
            while(!pq.empty()) pq.pop();
            for(int j = 1; j <= i; j++) pq.push(Node(j,f[j],0));
            int ht = h*12-st[i]; //总间隔数
//            cout << ht << "&&&" << endl;
            while(ht > 0)
            {
                Node tmp = pq.top();
//                cout << tmp.i << " " << tmp.f << " " << tmp.t << endl;
                if(tmp.f <= 0) break;
                pq.pop();
                res += tmp.f;
                tmp.f -= d[tmp.i];
                tmp.t ++;
                pq.push(tmp);
                ht--;
            }
//            cout << res << " * " << endl;
            if(res > ans)
            {
                ans = res;
                memset(ret, 0, sizeof(ret));
                while(!pq.empty()){
                    Node tmp = pq.top();
                    pq.pop();
                    ret[tmp.i] = tmp.t;
                }
//                cout << ht << "^^^ " << ret[1]<< endl;
                ret[1] += ht;
            }
        }
        for(int i = 1; i < n; i++) cout << ret[i] * 5 << ", ";
        cout << ret[n] * 5 << endl;
        cout << "Number of fish expected: " << ans << endl;
        cout << endl;
    }
    return 0;
}

9. HDU1789 再次作业
